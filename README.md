## 🧠 Pattern Cheatsheet

| Pattern | When to use |
|---|---|
| **Two Pointers** | Sorted array, pair sum, palindrome check |
| **Sliding Window** | Subarray / substring with constraint |
| **Binary Search** | Sorted search, minimise/maximise answer |
| **BFS / DFS** | Tree / graph traversal, shortest path |
| **DP** | Overlapping subproblems, optimal substructure |
| **Monotonic Stack** | Next greater element, span problems |
